# 1on1 Prep — 月次1on1の準備

メンバーとの定期1on1（月1〜2回 / 30分）の前に、**話すべき1〜2点を決めておく**ための準備フロー。

## 1on1の目的

- メンバーの **Will / Can / Must** の現在地を確認・対話
- 現在のPJ・働き方の摩擦を早期に拾う
- 個人としての成長と、組織として果たしたいことの接続を一緒に見る

**目的にしないもの**: タスクの進捗確認（それは定例で）、評価フィードバック（半期レビューで）

## 準備物

- 前回1on1メモ
- 直近1ヶ月の該当メンバーの稼働状況（参加PJ / Slack発言 / アウトプット）
- 該当メンバーのWill/Can/Must記述（`../_shared/stakeholder-map.md`）
- ダッシュボードのMemberセクション

## AIプロンプト例

### 1on1事前ブリーフ生成

```
以下の入力から、[メンバー名] との1on1ブリーフを生成して:

- 前回1on1メモ: [リンク]
- 直近1ヶ月の参加PJ: [リスト + 役割]
- 直近のSlack発言（公開チャンネルのみ）: [リンク]
- 現在のWill/Can/Must記述: [リンク]

出力フォーマット:
## メンバー: [名前]

### 前回からの変化
- [気づいた変化2-3点]

### 観察された動き
- ハイライト: [今月よく動いていたこと]
- 気になる点: [稼働偏り / トーン変化 / 静かになった領域]

### Will/Can/Must への問い
- Will: [今の仕事はWillに沿っているか / ズレが起きていないか]
- Can: [新しく身についたものは / 伸ばしたい領域は]
- Must: [今期Mustとして自覚していることは]

### 話すべき1-2点（推奨）
- [1: 一行 + 聞く問いのドラフト]
- [2: 一行 + 聞く問いのドラフト]

### 言わない方がよいこと
- [評価ニュアンスを含む話題 / タスクの進捗指摘 / 他メンバーとの比較]
```

### 問いの精緻化

```
1on1ブリーフ [リンク] の「話すべき1-2点」を、
- 開いた問い（Open Question）形式に変換
- 相手が3秒以上考える深さに
- 過去 / 現在 / 未来 の時間軸を1つは含める
- 「正解」を仕込まない（誘導しない）

例:
× 「最近忙しすぎない？稼働下げる？」（誘導）
○ 「先月の働き方、振り返ってどう感じている？」（開いた問い）
```

## 1on1中の作法

1. **冒頭5分**: 仕事以外の話（雑談を尊重する）
2. **次15分**: 準備した問い1-2点を中心に、相手が話す8割
3. **次5分**: メンバーから「話したいこと」を聞く（用意していなかったテーマを歓迎）
4. **最後5分**: 次回までに何を持ち越すか、お互いに1つずつ

## 1on1後の処理

```
1on1のメモ [本文] から、以下を抽出:
- 出てきたWill/Can/Mustの更新候補（あれば）
- 持ち帰り事項（自分側 / メンバー側）
- 次回1on1で扱うべき継続トピック

更新候補は ../_shared/stakeholder-map.md と ai-dashboard.html の Member セクション に
反映するための diff 案を生成して。
```

## ダッシュボード反映

Will/Can/Must の更新は、必ず以下に同期:

- `../_shared/stakeholder-map.md` の該当行
- `ai-dashboard.html` の Member セクション該当カード

## Push文テンプレ

1on1自体は基本DMで完結するため、チャンネル投稿は不要。
ただし、組織レベルの示唆が出た場合は以下:

```
:speech_balloon: People Mgmt — 共有

▼ トピック
[個人特定情報を伏せた抽象化された示唆]

▼ 検討事項
- [チームに関わる判断が必要なら]

▼ 該当者の同意
取得済み / 取得不要（抽象化のため）
```

## 注意

- **メモはメンバーと共有可能な範囲で書く** — 後から見せられないことを書かない
- **AIに渡すコンテキストに個人名・評価ニュアンスを長期保存しない** — 「処理後忘却」を指示する
- **頻度は固定しない** — 月1〜2回を基本とし、必要に応じて増減
- **キャンセルは双方OK** — どちらかが体調・心境的に難しい時は無理しない
- **DH本人の1on1相手** — DH自身も誰かと1on1を持つ。経営者は孤独を組織に持ち込まない
