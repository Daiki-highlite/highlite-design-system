# Glossary — 社内用語・略語

Highlite社内で頻出する用語の定義集。Claudeはこの定義を**基準**として使い、外部に出すときは「外部表現」列を採用する。

## フェーズ用語

| 社内表現 | 定義 | 外部表現 |
|---|---|---|
| **Ore（鉱石）** | リード獲得〜提案までのフェーズ | リード / 商談前 |
| **Lite（磨き、ラピダリスト）** | 受注後デリバリー | プロジェクト遂行中 |
| **Crysta（結晶）** | 検収後のナーチャリング | 既存顧客との関係性継続 |
| **lapidarist** | 宝石研磨師。Liteフェーズの担い手の比喩 | （訳さず使うか「ストーンカッター」） |

## プロセス略語

| 略語 | 定義 |
|---|---|
| **BANTC** | Budget / Authority / Need / Timing / Competitor。Discoveryでの聞き出し軸 |
| **Disco Note** | Discovery後にまとめる商談メモ |
| **RACI（簡易版）** | 主担当 / レビュー / 情報共有 の3区分（一般的なRACIを小規模組織向けに簡略化） |
| **WCM** | Will / Can / Must。メンバーの役割設計フレーム |

## チャンネル名

| チャンネル | 用途 |
|---|---|
| `#ore-リード・アサイン情報-cc` | Ore全フェーズ + アサイン |
| `#lite-lapidaristデリバリー-cc` | Lite運用全般 |
| `#lite-請求・入金-cc` | 請求・入金 |
| `#cry-ナーチャリング・イベント-cc` | Crysta全フェーズ |

`-cc` サフィックスは「Cowork Channel」の意。新規チャンネル作成時もこの命名規則に従う。

## 役職・ロール

| 表記 | 定義 |
|---|---|
| **社員** | フルタイム雇用 |
| **パートナー** | 業務委託（個人・法人含む） |
| **主担当** | RACIの Responsible。意思決定と実装の両方を持つ |
| **レビュー** | RACIの Accountable。最終承認 |
| **情報共有** | RACIの Informed。結果の通知のみ |

## 伏字ルール（社外向け）

以下のものは **明示的な許諾なく外部公開しない**:
- 顧客名（Crysta 01「実績開示の許可」を経たもののみ実名OK）
- 案件金額・利益率
- 個別メンバーの稼働率・評価
- 未公開の事業計画

伏字表現の例:
- 「[クライアント名]」「A社」「業界・規模感は[XX]」
- 「契約金額: 数百万円規模」など レンジ表現

## 禁則用語

外部・社内問わず使わない:
- 「絶対」「100%」「保証」（過度な約束）
- 「他社にはできない」「業界初」（裏付けなしの優位主張）
- ネガティブ評価の固有名表現（具体名を出した競合批判）
