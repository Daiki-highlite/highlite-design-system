### LP評価（Quartz Setup）

**URL**：https://highlite.co.jp/lp/quartzsetup/

#### LP構成
- ヒーロー：「デザインに、時間を取られない経営へ。」
- コンセプト：「外注でも属人化でもない、第三の選択肢」
- プラン：Sharp（40万/2週間）/ Clear（80〜150万/3ヶ月）/ Prism（200万〜/4ヶ月〜）
- CTA：無料30分ヒアリング（フォーム2箇所）

#### 良い点
- ヒーローメッセージが明快
- 3プランの価格・期間・成果物が明示されている
- 無料ヒアリングで問い合わせハードルが低い
- FAQが主要な懸念点をカバー

#### 課題と対応方針

| 課題 | 対応方針 |
|------|---------|
| 「社内にデザイン機能を構築する」という解決策がファーストビューで伝わらない | LPブラッシュアップで改善 |
| ターゲット企業の解像度が低い（業種・規模・状況が不明） | DL資料で補填 |
| 導入後の具体的な成果イメージがない | DL資料で補填 |
| 実績・社会的証明がない | 現状は対応困難。今後の実績蓄積で対応 |
| SaaS文脈への訴求がない | LPブラッシュアップで改善検討 |
| 提供者の信頼性情報がない | Highlite本サイトへ誘導することで対応済み |

### DL資料の設計

#### 目的
- LPで離脱しそうな「まだ迷っている層・情報収集層」をメールアドレスで捕捉する
- ターゲット解像度と成果イメージの不足をLPの外で補填する

#### LP上のCTA構成（2本立て）
```
[無料ヒアリングを予約する]  ←  今すぐ相談したい層
[まず資料を見てみる]       ←  まだ迷っている層・情報収集層
```

#### 資料の構成（確定）
1. 自己診断チェックリスト（「うちの会社に必要か」を判断）
2. ターゲット企業の典型状態（ビフォー）
3. 導入後の状態（アフター）
4. Quartz Setupのアプローチ・タイムライン
5. 外注vs内製化のコスト比較
6. よくある失敗パターン
7. LP内診断へ誘導（「詳しく診断してみる →」）
8. ネクストアクション（問い合わせ・ヒアリング予約）
