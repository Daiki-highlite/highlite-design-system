FYI

よくお寄せいただく問い

Q. どのプランから始めるべきでしょうか。 +
価格ではなく、いま必要な深さ・得たい成果・組織のフェーズで選ぶことをおすすめしています。大きな判断を控えているなら Sharp、AI と判断基準の土台を整えたいなら Clear、デザインシステムとナレッジまで実装したいなら Prism が適しています。無料相談で御社に最適なプランをご提案します。
Q. Claude Design とは何ですか。 +
Claude Design は、Anthropic 社が提供するデザイン特化の AI ツールです。デザインシステムの構築から Web ページや UI の制作まで、テキスト指示や対話を通じてデザイン領域のアウトプットを高品質かつ高速に生成できます。Quartz Setup の Clear / Prism では、この Claude Design をブランドガイドラインに沿って活用するためのプロンプト設計、判断基準の整備、チームで再現可能なワークフローまで体系的に組織へ実装します。
Q. Sharp から Clear・Prism へ移行することはできますか。 +
はい。Sharp で得られたコアの言語化を土台に、Clear や Prism へ段階的に進んでいただくケースが多くあります。Sharp の成果物はそのまま次フェーズの前提としてご活用いただけます。
Q. 難易度の高いデザイン制作も依頼できますか。 +
Webページやアプリ UI 全体など、難易度の高いデザイン制作については、いずれのプランでも Highlite が別途費用で制作を承ります。プランの中で基準と仕組みを整えつつ、必要に応じて Highlite が制作パートナーとしてご支援します。
Q. オンラインでの実施は可能ですか。 +
はい。Sharp は完全オンラインでも実施可能です。Clear・Prism はハンズオンや運用設計の場面で対面を交えた実施を推奨しておりますが、ご要望に応じてフルオンラインでも承ります。
Q. プラン終了後のフォローはありますか。 +
Clear・Prism では、プラン終了後のチャットサポートと定期的な振り返りセッションをご提供しています。運用の中で磨き続けるところまで、伴走することを大切にしています。
